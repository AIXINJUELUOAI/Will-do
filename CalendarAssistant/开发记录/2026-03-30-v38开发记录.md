# 2026-03-30 开发记录 (v38版本)

## 一、本次实现范围

1. **悬浮窗输入框 UI 重构**：弃用 `OutlinedTextField`，改用 `Surface + BasicTextField` 实现现代胶囊化设计
2. **日程/便签模式切换**：输入框左侧新增模式切换按钮，支持日程和便签两种输入模式
3. **发送按钮交互优化**：实心圆形按钮 + 颜色动画，始终显示圆圈图标
4. **图标替换**：使用项目内置矢量图标替换 Material 默认图标

---

## 二、详细改动

### 2.1 输入框 UI 重构

**旧方案**：`OutlinedTextField` + `trailingIcon` 内嵌图片和发送按钮

**新方案**：
- `Surface(RoundedCornerShape(26.dp))` 构建悬浮胶囊底座
- `BasicTextField` 无边框输入，支持多行（maxLines=4）
- `tonalElevation = 0.dp` 去除 Material 3 默认染色，保持纯净白底
- `shadowElevation = 8.dp` 增强悬浮立体感
- 固定高度 `60.dp`，与日程卡片高度一致
- 左右边距 `28.dp`，远离屏幕边缘

**布局变化**：
```
旧: [OutlinedTextField                     [📷][➤]]
新: [📅 | 一句话安排日程...          | 📷 | ⬆]
```

### 2.2 日程/便签模式切换

- 新增 `FloatingInputMode` 枚举（`SCHEDULE` / `NOTE`）
- 左侧按钮：`ic_stat_event`（日程）/ `ic_stat_note`（便签），带 `Crossfade` 切换动画
- 背景色随模式变化：日程用主题色，便签用橙色（`0xFFF57C00`）
- `onManualSubmit` 回调新增 `isNote: Boolean` 参数
- placeholder 跟随模式变化："一句话安排日程..." / "一句话安排便签..."
- 便签具体业务逻辑暂未实现，参数仅透传

### 2.3 发送按钮优化

- **始终显示圆圈 + ArrowUpward 图标**
- 未输入时：`activeColor.copy(alpha = 0.15f)` 浅色背景 + `activeColor.copy(alpha = 0.6f)` 图标
- 有文字时：`activeColor` 实色背景 + 白色图标
- 颜色切换带 `tween(150)` 动画

### 2.4 图标替换

| 位置 | 旧图标 | 新图标 |
|------|--------|--------|
| 模式切换（日程） | emoji 📅 | `ic_stat_event` |
| 模式切换（便签） | emoji 📝 | `ic_stat_note` |
| 图片选择器 | `Icons.Rounded.Image` | `ic_stat_scan` |
| 发送按钮 | `Icons.Rounded.Send` → `Icons.Rounded.ArrowUpward` | 保持 `ArrowUpward` |
