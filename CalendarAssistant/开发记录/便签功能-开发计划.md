# 便签（Notes）功能开发计划

> 日期：2026-03-31
> 状态：规划中

## 一、功能概述

为 Will do 智能日历助手新增「便签」功能，允许用户快速记录灵感、待办、碎片化想法。

**核心设计决策**：
- 作为**实验室实验性功能**推出，默认关闭
- 复用 `MyEvent` 模型，通过 `tag = EventTags.NOTE` 数据隔离
- **不改 Room 数据库表结构**

## 二、交互设计

### 2.1 启用方式
设置 → 实验室 → 「启用便签」开关打开后，底栏从 2 Tab 变为 3 Tab。

### 2.2 底栏 Tab
| Tab | 图标 | 页面 |
|-----|------|------|
| 今日 | Home | 日程列表（今日+明日） |
| 便签 | StickyNote2 | 瀑布流便签页 |
| 全部 | List | 全部日程页 |

### 2.3 便签页（NotePage）
- `LazyVerticalStaggeredGrid` 2 列瀑布流
- 卡片：title（加粗）+ description（maxLines=5）
- 右上角 Checkbox 标记完成
- 长按/滑动删除
- 空状态提示

### 2.4 创建入口
- 悬浮窗输入框：左侧模式切换按钮（已有 `FloatingInputMode.NOTE`），切换为便签模式后输入即创建便签
- 便签页右上角 FAB 按钮（后续可加）

### 2.5 便签 Event 结构
```kotlin
MyEvent(
    title = "便签标题",
    description = "便签内容",
    startDate = LocalDate.now(),  // 兜底日期
    endDate = LocalDate.now(),
    startTime = "00:00",
    endTime = "23:59",
    tag = EventTags.NOTE,
    skipCalendarSync = true,
    reminders = emptyList()
)
```

## 三、隔离策略

便签必须从以下链路中完全隔离：

| 链路 | 拦截方式 |
|------|---------|
| 日历同步（CalendarSyncManager/V2） | `eventsToSync` 过滤 `tag != NOTE` |
| 胶囊/超级岛（CapsuleStateManager） | `allEvents` 过滤 `tag != NOTE` |
| 每日播报（DailySummaryReceiver） | 事件列表过滤 `tag != NOTE` |
| 自动归档（AppRepository） | `toArchiveEvents` 过滤 `tag != NOTE` |
| 规则引擎（RuleMatchingEngine） | `resolvePayload()` 提前返回 null |
| EventPresenter | 便签极简渲染分支 |

## 四、实施阶段

### 第一阶段：数据层
1. `EventTags.NOTE` 常量
2. `MySettings.noteEnabled` 开关字段
3. `AppRepository` 拆分 `scheduleEvents` / `notesFlow`（用 `stateIn`）
4. `addEvent()` 中 NOTE 自动设置 `skipCalendarSync=true`、`reminders=empty`
5. `autoArchiveExpiredEvents()` 排除 NOTE
6. `MainViewModel` 暴露 notes 数据

### 第二阶段：防御性拦截
1. CalendarSyncManager / V2 过滤
2. CapsuleStateManager 过滤
3. DailySummaryReceiver 过滤
4. RuleMatchingEngine 兜底
5. EventPresenter 兜底渲染

### 第三阶段：UI 层
1. LaboratoryPage 便签开关
2. 底栏 Tab 扩展（2↔3 动态切换）
3. NotePage 瀑布流页面
4. 悬浮窗便签创建确认
5. 主页/全部页过滤便签

### 第四阶段：规则引擎兼容
已在第二阶段覆盖。
